What is JavaScript?

JavaScript is an object-oriented scripting language that is a cross-platform that operates in a host environment such as a web browser. JavaScript also contains Arrays, Dates, math, a set of language elements like operators, control structures and statements. JavaScript is commonly used to created interactive effects within web browsers.
